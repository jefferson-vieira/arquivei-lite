## Prerequisites

- node >=14.17.0
- yarn ^1.22.5

## Install

```sh
yarn install
```

## Run

### Development

```sh
yarn dev
```

### Production build

```sh
yarn build
yarn start
```

#### With [Docker](https://www.docker.com/)

```sh
docker build . -t arquivei-lite
docker run -p 3000:3000 -d arquivei-lite
```

Or with docker-compose

```sh
docker-compose up -d
```

Then open [http://localhost:3000](http://localhost:3000) to see the app.<br>
